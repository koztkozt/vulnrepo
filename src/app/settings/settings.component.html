info Template Placeholders Guide

Available Placeholders:

  • {{ '{' }}report_name{{ '}' }} - Report title
  • {{ '{' }}report_id{{ '}' }} - Report ID
  • {{ '{' }}scope{{ '}' }} - Testing scope
  • {{ '{' }}total_issues{{ '}' }} - Total number of issues
  • {{ '{' }}critical_count{{ '}' }} - Number of critical issues
  • {{ '{' }}high_count{{ '}' }} - Number of high severity issues
  • {{ '{' }}medium_count{{ '}' }} - Number of medium severity issues
  • {{ '{' }}low_count{{ '}' }} - Number of low severity issues

Loops (for repeating data):

  • {{ '{' }}#issues{{ '}' }} ... {{ '{' }}/issues{{ '}' }} - Loop through all issues
  • Inside issues loop: {{ '{' }}title{{ '}' }}, {{ '{' }}severity{{ '}' }}, {{ '{' }}description{{ '}' }}, {{ '{' }}proof_of_concept{{ '}' }}
  • {{ '{' }}#changelog{{ '}' }} ... {{ '{' }}/changelog{{ '}' }} - Loop through changelog entries
  • {{ '{' }}#researchers{{ '}' }} ... {{ '{' }}/researchers{{ '}' }} - Loop through researchers
